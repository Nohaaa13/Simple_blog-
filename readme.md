## Installation

```
git clone https://github.com/Nohaaa13/Simple_blog-.git
cd Simple_blog
composer install
cp .env.example .env
write in env your db setting
php artisan key:generate
php artisan migrate
composer dump-autoload
php artisan db:seed
```
